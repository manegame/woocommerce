# PWR Boilerplate

vue (+ wp)

Based on vue-cli/webpack

## Build Setup

``` bash
# install dependencies with Yarn (yarnpkg.com)
yarn

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
